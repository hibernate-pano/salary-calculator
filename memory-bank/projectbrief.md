# 项目简介：工资计时器

## 是什么

一个 iOS「摸鱼神器」。上班时间里，用户看着自己赚的每一秒钱在屏幕上实时跳动。

核心体验：填月薪 + 上下班时间 → 主页看今日/本月/年度收入实时累计。

## 真实定位（重要）

本项目历史上有一份虚构的产品文档（把它描述成全功能 HR 薪资工具：社保/公积金/个税/iCloud 同步/多方案对比），但代码从未实现这些。2026-06 已把文档与代码对齐到真实产品——「实时工资计数器」。

## V1 范围（已完成）

- 实时工资主界面（今日/本月/年度，每秒跳动）
- 设置页（月薪、上下班、午休、工作日）
- 首次启动引导页
- 本地持久化（SwiftData）

## V1 明确不做

- 联网节假日/调休（模型保留，数据源未接）
- 加班倍率、税费
- 桌面 Widget、统计图表、数据导入导出

这些相关代码已移至 `archive/`，移出编译路径但保留备用。

## 技术决策

- SwiftUI + SwiftData（`@Model` + `@Query`），不用单独的服务层/Repository
- `TimelineView(.periodic)` 驱动每秒刷新
- 计算引擎 `SalaryCalculator` 与界面解耦，纯函数、可测试
